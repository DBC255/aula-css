Introducão ao CSS

1. HTML e CSS: Estrutura e estilo

O HTML cria a Estrutura da página (títulos, parágrafos, imagens), enquanto o CSS define como tudo vai aparecer -cores, fontes, tamanhos, bordas, espaçamentos e posicionamento.

HTML

<h1>Teste 1</h1>
<p>Teste 2 </p>

CSS

h1 {color: blue; }
p { color:gray; }

Observação: O navegador interpreta primeiro o HTML, criando a estrutura da página, e depois aplica o CSS, alterando a a aparência visual

2. Como aplicar o CSS

Existem três maneiras de aplicar o CSS

2.1 CSS inline

você escreve o estilo direto na tag HTML usando o atributo style

<p style="color: red;">Texto vermelho</p>
Texto vermelho Utilizando CSS Inline

Quando usar: apenas para testes rápidos. evitem em projetos grandes, porque dificulta a manutenção

2.2 Css Interno

você coloca as regras dentro da tags <style> no <head> do HTML.

<style>
p { color: blue }
</style>

Texto azul usando o CSS Interno

Quando utilizar: útil para paginas pequenas ou para testar rapidamente sem criar arquivos extras

2.3. CSS Externo

O estilo fica em um arquivo separado, como style.css. Depois você vincula com <link rel="stylesheet" href="style.css">

é a forma mais organizada e usada em projetos reais, pois mentém o código limpo e reutilizável.

3. Sintaxe básica do CSS

Uma regra CSS tem três partes:

  • selector: diz ao navegador quais elementos estilizar
  • propriedade: o que você quer mudar (cor, tamanho, etc...)
  • valor:o detalhe do estilo aplicado
p { color: green; }

Texto verde aplicado com CSS

9. Box model

Todo elemento HTML é uma "caixa" com 4 camadas:

  • Content: O conteudo (texto, imagem, video )
  • Padding: espaaço interno entre conteudo e borda
  • Border: borda ao redor do padding
  • Margin: espaço externo que separa a caixa de outros elementos

9.1. Margin

define o espaaço externo pode ser individual(top, botton, left, right)

margem externo de 30 px.

9.2. Padding

Define o espaço interno entre o conteudo e a borda Facilita a leitura e evita que o conteúdo fique colado à borda .

padding interno de 30 px.

9.3. Combinando (margin e padding)

Você pode ajustar padding, border e margin juntos para criar layouts
